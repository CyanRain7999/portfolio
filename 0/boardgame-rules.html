一、基础规则（主世界 · 本篇）

建议将纸质规则书中关于 Lv1 对局结构、基本名词解释、资源与回合流程的章节搬到这里。

1. 对局目标与胜负条件

【示例占位】这里写清楚一局游戏的获胜条件、失败条件，以及是否存在平局、特殊结算等情况。

2. 牌库构筑与卡牌种类

【示例占位】说明开启对局前如何构筑卡组、每个卡包的基础数量要求，以及“政策 / 装备 / 单位”等基本类别。

3. 回合流程

  1. 回合开始阶段（重置、维持、抽牌等）
  2. 主要阶段（部署单位、使用政策、装备）
  3. 战斗阶段或结算阶段（如果有的话）
  4. 回合结束阶段（弃牌上限、持续效果结算等）

4. 常见关键字

对“核心单位”、“雇佣单位”、“泛用”、“控制”、“经济”等关键字做简要解释， 详细解释可以链接到「规则与词条」页面的对应词条。

二、扩展规则（DLC1：新教徒 / 森妖怪）

这里集中描述当使用 DLC1 卡包时，额外增加的规则、关键字与对基础规则的修改。

1. 使用 DLC 卡包的对局准备

【示例占位】是否需要替换基础卡、增加某些初始资源、改动起始手牌等。

2. 新教徒相关规则

【示例占位】写清楚新教徒卡包引入的特殊状态、仪式流程、转换条件等。

3. 森妖怪相关规则

【示例占位】写清楚“森林区域”、“环境效果”等与森妖怪有关的特殊机制。

三、镜世界规则（DLC2：鸿园等）

当对局进入镜世界时，采用何种额外流程、时空/平行世界相关的特殊条目在此处说明。

1. 进入镜世界的条件

【示例占位】例如某种计数器达到一定值时，对局“翻面”进入镜世界。

2. 镜世界中的通用变化

  • 资源规则是否改变
  • 某些关键字是否获得新的含义
  • 部分卡牌在镜世界时是否有替代效果

3. 鸿园等卡包的专属补充

【示例占位】针对鸿园等卡包特别规定的镜世界互动。

A：说明处理顺序，例如“主动玩家优先”或“按时间顺序结算”等。
